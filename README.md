# newsclip-feed-healthcheck

Lightweight URL healthcheck for Newsclip pages (status, latency, bytes).

A lightweight script designed for simple pipelines.

## Why this exists
- Reduce friction when prototyping pipelines and monitoring checks.
- Support quick checks and repeatable automation without heavy dependencies.

## More from Newsclip
- [Newsclip](https://newsclip.com/)
- [Newsclip Terms](https://newsclip.com/terms/)
- [Newsclip Privacy](https://newsclip.com/privacy/)

## Usage
Example:

```bash
npm install
npm run start -- https://newsclip.com/
```
